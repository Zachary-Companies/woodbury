# LinkedIn Selectors & Element Locations

## Navigation (Top Bar)
| Element | Description | Notes |
|---------|-------------|-------|
| LinkedIn Logo | Top left | Links to feed |
| Search Bar | Top center | Easily mis-clicked |
| Home | Nav icon | Goes to /feed/ |
| My Network | Nav icon | Connections |
| Jobs | Nav icon | Job listings |
| Messaging | Nav icon | Direct messages |
| Notifications | Nav icon | Activity alerts |
| Profile/Me | Nav icon (right) | Account menu |

## Start a Post Section (Center Column, Top)
| Element | Description | Notes |
|---------|-------------|-------|
| Profile picture | Left side of input | Small circular avatar |
| Start a post input | Rounded rectangle | White/light gray, clickable |
| Video button | Below input, left | Opens post dialog with video |
| Photo button | Below input, middle | Opens post dialog with photo picker |
| Write article | Below input, right | Opens article editor (different flow) |

## Post Creation Modal (When Open)
| Element | Description | Notes |
|---------|-------------|-------|
| Modal container | Overlay/dialog | Covers feed |
| Text area | Large input | "What do you want to talk about?" |
| Media buttons | Bottom of modal | Add photos/videos/documents |
| Audience selector | Near profile pic | Who can see |
| Post button | Top right, blue | Publishes the post |
| Close/X button | Top right corner | Dismisses modal |

## Coordinate Estimates (2056x1329 screen)
- Browser chrome height: ~125px
- Center column X range: 400-1100px
- Start a post input Y: 200-320px (from screen top)
- Video/Photo buttons Y: 340-380px

## Login Detection
| Indicator | How to Check | Notes |
|-----------|-------------|-------|
| Logged in | Profile avatar in nav, "Start a post" visible | |
| Not logged in | Login form, "Join now" button | Redirect to login page |
